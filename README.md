# stockProject
주식 분석 웹사이트
<h1> 프로젝트 소개</h1>
인원: 1인 개발<br>
개발 기간: 2-3주<br>
배포방법: 미정<br>
주요 기능<br>
 1. 백테스팅<br>
 2. 커뮤니티<br>

<h1> 사용한 기술</h1>
front-end: html, css, js<br>
back-end: django<br>
개발 환경: visual studio code<br>
db: mysql(직접 sql구문 작성보다는 orm으로 사용)<br>

<h1> 사용한 오픈소스 및 데이터</h1>
open api: yfinance<br>
lib: numpy, matplotlib, mpld3 등<br>
데이터: krx - 주식 기본 정보<br>

<h1>프로젝트 구조</h1>
<h1>앱별 상세 기능 및 구현방법</h1>

user - 회원 관련<br>
